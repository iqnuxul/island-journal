# Island Journal 🌿

A cozy, Animal-Crossing-inspired slot machine for daily reflection. Pull the lever, get two journaling prompts, write what comes up.

## Features
- Cute island scene (sky, clouds, rolling grass hills)
- AC-style chunky outlined UI with red striped roof
- Two prompt reels: REFLECT (deep) + CHECK IN (everyday)
- Notebook with one writing box per prompt
- Auto-saves to browser (survives tab close)
- Download today's entry as `.txt`
- Pure HTML/CSS/JS — no build step

## Run locally
```bash
python3 -m http.server 5175
```
Then open http://localhost:5175

## Deploy
```bash
vercel
```
